== Description ==

LegalPages Pro helps website owners generate essential legal pages like Privacy Policy, Terms, Cookies Policy, and more — along with a cookie consent banner.

Whether you're running a blog, store, or business site, this plugin helps you create:
- Privacy Policy  
- Terms & Conditions  
- Disclaimer  
- Cookies Policy  
- GDPR Notice  
- Refund Policy  
- DMCA Page  
- Cookie Consent Banner  

All pages are editable and customizable with your business details.

== Features ==

- One-click generation of legal pages  
- Admin panel for business info  
- Adds pages to WordPress menus  
- Cookie consent popup  
- Standard WordPress pages for full theme support

== Installation ==

1. Upload the plugin to `/wp-content/plugins/` or install via WordPress Plugin Installer.  
2. Activate the plugin from the Plugins menu.  
3. Go to **LegalPages Pro** in the admin menu.  
4. Set your info and choose pages to generate.  
5. Click **Generate Pages** — done!

== FAQ ==

= What is LegalPages Pro? =
A plugin to auto-generate legal pages like Privacy Policy, Terms, DMCA, and more.

= Are the pages editable? =
Yes, they are regular WP pages and can be customized freely.

= Does this help with GDPR? =
Yes, it includes GDPR text and a cookie consent popup.

= Can I choose which pages to generate? =
Yes, select only the ones you want.

= Is the plugin lightweight? =
Yes, it’s optimized and fast.
